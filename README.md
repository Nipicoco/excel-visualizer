### Install with npm or yarn

1. **Clone this repository**  
   ```sh
   $ git clone https://github.com/Nipicoco/excel-visualizer.git
   ```

2. **Go into the repository**  
   ```sh
   $ cd excel-visualizer
   ```

3. **Install dependencies**  
   ```sh
   $ npm install
   # or
   $ yarn install
   ```

4. **Start the development server**  
   ```sh
   $ npm run dev
   # or
   $ yarn dev
   ```
